# Slider

Using [https://range-slider.toolcool.org/](https://range-slider.toolcool.org/)

#### Params

| Key                       | Value                      | Default           |
|:--------------------------|:---------------------------|:------------------|
| secondaryLabel       | Secondary label for slider | `""`              |
| unit     | unit label                 | `""`                |
| step              | slider step                | `1` for integer   |
