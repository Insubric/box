# Multi

Rendering lists using multiple time the same widget

#### Params

| Key        | Value                                         | Default   |
|:-----------|:----------------------------------------------|:----------|
| widget     | widget name                                   | Mandatory |
| widgetFieldOpts | widget options                                | `null`    |
| incrementalParams    | list of params to overwrite by the list index | `[]`      |
