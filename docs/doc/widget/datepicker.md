# Datepicker

Calendar popup widget for date object, based on [Flatpickr](https://flatpickr.js.org/)


#### Supported types
- **Date**

#### Params

| Key          | Value             | Default           |
|:-------------|:------------------|:------------------|
| fullWidth    | show the picker at 100% width | `false`  |
