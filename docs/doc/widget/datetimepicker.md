# DateTimePicker

Calendar popup widget for timestamp object, based on [Flatpickr](https://flatpickr.js.org/)


#### Supported types
- Timestamp

#### Params

| Key          | Value             | Default           |
|:-------------|:------------------|:------------------|
| fullWidth    | show the picker at 100% width | `false`  |
